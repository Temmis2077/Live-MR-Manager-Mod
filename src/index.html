인트로 자동 건너뛰기

유튜브 영상 앞에 뮤비 형식 인트로 등 노래와 무관한 구간이 있는 곡은, 가사 싱크 탭에서 "보컬 시작 지점"을 지정해두면 재생 시 그 지점으로 자동으로 건너뜁니다. 지점을 지정하지 않은 곡은 평소처럼 처음부터 재생됩니다.

곡 정보 관리

곡 정보 일괄 수정

라이브러리 곡 정보를 표 형태로 한꺼번에 수정합니다.

엑셀(CSV) 가져오기 / 보내기

곡 목록을 CSV·엑셀로 내보내거나 가져옵니다. 양식 받기로 형식을 확인하세요. 같은 곡은 갱신되고 새 곡은 추가됩니다.

데이터 백업 및 복원 (JSON)

곡 목록을 파일로 백업·복원합니다. 복원 시 기존 곡은 유지됩니다.

멜로밍 노래책

멜로밍 노래책

가져오기는 멜로밍 노래책 → 앱(병합·갱신), 보내기는 앱 → 멜로밍 노래책입니다. 보내기는 우측 상단 「멜로밍 로그인」이 필요합니다. 없는 아티스트·카테고리는 보내기 시 자동으로 등록합니다.
