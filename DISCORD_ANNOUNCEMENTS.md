## v0.5.0 (2026-07-05)

**Live MR Manager v0.5.0** 업데이트가 나왔습니다.

**이번에 새로워진 점**
- **멜로밍 노래책 가져오기 / 보내기**를 다시 쓸 수 있습니다
  - 설정 → 멜로밍 노래책 → **가져오기**: 멜로밍 → 앱
  - **보내기**: 앱 → 멜로밍 (우측 상단 **멜로밍 로그인** 필요)
- 보낼 때 **KEY, BPM, 가사, 난이도·숙련도**도 함께 맞춰 줍니다
- 없는 **아티스트·카테고리**는 보내기할 때 자동으로 등록됩니다
- 곡 **정보 수정**에서 난이도·숙련도를 **별 클릭**으로 고를 수 있습니다

**업데이트 방법**
1. 아래 **다운로드**에서 Windows 설치 파일 받기
2. 설치 후 기존처럼 사용 (라이브러리·설정은 그대로 유지)

궁금한 점은 **#질문-답변**에 남겨 주세요.
