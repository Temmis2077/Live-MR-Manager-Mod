# 🎙️ Live MR Manager(Beta) v0.4.12 — 사용자 매뉴얼

**Live MR Manager(Beta)**는 AI 기술을 활용해 어떤 노래든 고품질 MR로 분리하고, 라이브 환경에서 실시간으로 키(Pitch)와 템포(Tempo)를 조절할 수 있는 전문적인 하이브리드 오디오 도구입니다.

---

## 🚀 1. 프로젝트 시작하기

### 🌓 앱 테마 설정 (4종)
`설정 > 앱 테마`에서 **미드나잇(다크) / 아이보리(라이트) / 분홍 / 하늘** 테마를 즉시 전환할 수 있습니다.
- 다크/라이트뿐 아니라 퍼포먼스 분위기에 맞는 컬러 테마를 빠르게 선택할 수 있습니다.
- 선택한 테마는 자동 저장되어 앱 재실행 후에도 유지됩니다.

### ⚙️ AI 모델 설치 및 준비
프로그램을 최고의 성능으로 사용하려면 최초 1회 모델 설치가 필요합니다.
1. 왼쪽 사이드바에서 **[설정]** 메뉴를 클릭합니다.
2. **[모델 다운로드]** 버튼을 클릭하여 AI 분리 엔진을 설치합니다.
3. 왼쪽 상단의 상태 배지가 `Online (Ready)`로 변경되면 모든 준비가 완료됩니다.

> [!TIP]
> **NVIDIA 그래픽카드** 사용자라면 `CUDA`와 `cuDNN`을 설치하세요. 설정 페이지 하단에 감지 시 안내가 표시되며, 설치 시 분리 속도가 최대 10배 이상 빨라집니다.

## 🎵 2. 미디어 관리 (음악 추가)

노래를 라이브러리에 추가하는 방법은 세 가지이며, 추가된 곡은 자동으로 썸네일이 생성되어 저장됩니다.

### 🍱 라이브러리 (Library)
- 추가된 모든 곡을 관리하는 메인 화면입니다. 
- 상단 **그리드/리스트/버튼** 모드를 통해 원하는 뷰로 전환할 수 있습니다.
- **[목록 관리]** 버튼을 눌러 엑셀 스타일의 일괄 편집 도구를 열 수 있습니다.

### 📺 유튜브 (YouTube)
1. 유튜브 영상 주소(URL)를 입력창에 붙여넣습니다.
2. **[정보 가져오기]** 버튼을 누르면 버튼에 로딩 표시가 적용되며, 제목/아티스트/썸네일을 자동 분석해 라이브러리에 추가합니다.
3. 동일한 YouTube 영상(동일 video ID)은 중복 등록이 자동으로 차단됩니다.

### 📁 내 파일 (Local Files)
- 내 컴퓨터에 있는 음원 파일(`MP3`, `WAV`, `FLAC`, `M4A` 등)을 화면 중앙으로 **드래그 앤 드롭**하여 즉시 추가할 수 있습니다.

## 🤖 3. AI 음원 분리 (AI Processing)

### 🎼 MR 분리 실행
1. 라이브러리에서 곡 카드 위로 **마우스 우클릭**을 합니다.
2. **[MR 분리]**를 선택하면 분리가 시작됩니다.
3. 분리 작업은 **[AI 프로세싱]** 메뉴에서 진행률을 실시간으로 확인할 수 있으며, 분리 도중에도 다른 곡을 재생하거나 탐색할 수 있습니다.

### 🧪 모델 선택 (설정)
- **Kim Vocal 2**: 보컬 추출 및 일반적인 분리에 최적화된 고속 모델입니다.
- **Inst HQ 3**: 반주(Inst)의 품질을 극대화하여 실제 공연용 MR에 적합한 고품질 모델입니다.

## 🎛️ 4. 라이브 컨트롤 (하단 도크)

재생 중 실시간으로 모든 음향 요소를 제어할 수 있습니다.

| 기능 | 설명 |
| :--- | :--- |
| **PITCH (피치)** | 키를 -12 ~ +12 반음 단위로 조절합니다. |
| **TEMPO (템포)** | 속도를 0.5x ~ 2.0x 범위로 조절합니다. |
| **VOLUME (볼륨)** | 마스터 음량을 조절합니다. (최대 120%) |
| **보컬 스위치** | 분리된 곡 재생 시 보컬의 ON/OFF를 300ms 페이딩과 함께 실시간 전환합니다. |
| **보컬 밸런스** | '보컬' 텍스트를 클릭하면 반주와 보컬의 음량 비율을 정밀하게 믹싱할 수 있는 팝오버가 열립니다. |

> [!IMPORTANT]
> **수치 직접 입력**: 피치, 템포, 볼륨 우측의 **수치(숫자)**를 클릭하면 키보드로 정확한 수치를 직접 입력할 수 있습니다.

## 🎨 5. 스튜디오 (고급 기능)

### ✍️ 가사 싱크 (Lyric Sync)
가사가 없는 곡에 실시간으로 가사를 입히는 도구입니다.
1. 가사 텍스트를 입력하고 오디오 파형(Waveform)을 확인합니다.
2. 음악을 들으며 특정 시점에 `Space`나 클릭을 통해 **탭-투-싱크(Tap-to-Sync)** 방식으로 싱크를 맞출 수 있습니다.
3. 필요 시 **강제 정렬(Forced Alignment)**을 실행해 단어/라인 타임코드를 자동으로 재계산하고, 페널티 튜닝으로 결과를 즉시 미세조정할 수 있습니다.
4. 작업 도중 취소하면 정렬 작업이 즉시 중단되며, 저장된 `.lrc`는 로컬 파일/유튜브 캐시 경로에 맞춰 자동 로드됩니다.
5. 작업 완료 후 라이브러리에서 곡을 재생할 때 우측 **가사 드로어(LYRICS)**를 통해 가사가 출력됩니다.
6. 테마에 맞춰 파형, 세그먼트, 가사 카드 대비가 자동으로 조정되어 어느 모드에서도 시인성을 유지합니다.

### 📺 OBS 오버레이 (OBS Overlay)
인터넷 방송(OBS) 송출을 위해 현재 재생 중인 곡 정보를 실시간 HUD로 제공합니다.
1. **[OBS 오버레이]** 메뉴에서 배경 투명도, 배율, 둥글기, 폰트, 테마 색상을 커스터마이징합니다.
2. 곡 정보 오버레이와 가사 전용 오버레이 URL이 분리 제공되며, 각각 독립적으로 배치할 수 있습니다.
3. 생성된 **URL**(`http://localhost:14202/`, `http://localhost:14202/lyrics`)을 복사해 OBS의 브라우저 소스에 붙여넣습니다.
4. **상시 표시** 옵션을 켜면 곡이 멈춰있어도 화면 구성을 미리 확인할 수 있습니다.

> [!TIP]
> 메타데이터 모달의 **KEY/BPM 분석** 버튼으로 현재 곡의 키/템포를 자동 분석해 바로 반영할 수 있습니다.

## 📡 6. 시스템 관리 및 안전

### 🛡️ 방송 제원 보호 (Broadcast Safe)
AI 분리가 고사양 작업임을 감안해, 방송 중 시스템 부하를 최소화하는 기능입니다.
- 활성화 시 AI 연산 간격을 강제로 조절하여 OBS 등 방송 송출 프로그램의 프레임 드랍을 원천 방지합니다.

### 📂 데이터 및 백업
- 모든 데이터는 내 컴퓨터 로컬(`%LOCALAPPDATA%`)에 저장됩니다.
- **설정** 메뉴에서 **[목록 백업]** 및 **[목록 복원]** 기능을 통해 소중한 라이브러리 DB를 JSON 파일로 안전하게 관리할 수 있습니다.
- **곡 정보 관리자**에서 곡 메타데이터를 표 형태로 일괄 수정할 수 있으며, 열 너비/버튼 레이아웃이 저해상도에서도 읽기 쉽게 최적화되어 있습니다.

### 🎵 멜로밍 노래책 (설정)
1. **설정 → 멜로밍 노래책**에서 **방송 채널 주소**를 입력합니다.
   - **치지직**: `chzzk.naver.com/…` 또는 채널 ID
   - **SOOP(숲)**: `sooplive.co.kr/station/…`
   - **씨미**: `ci.me/@아이디` 또는 `@아이디`
   - **멜로밍 webPath**·숫자 채널 ID도 가능
2. **[채널 저장]** → **[연결 테스트]** → **[가져오기]** 순으로 노래책 메타를 Pull합니다. (로그인 불필요)
3. 채널 주소는 **내 PC DB에만** 저장됩니다. 앱을 새로 설치하면 비어 있습니다.
4. **「멜로밍 로그인」**(우측 상단)·**「멜로밍에 보내기」**는 OAuth 연동 완료 전까지 **「개발 중입니다.」** 안내가 표시됩니다.

### ✏️ 곡 정보 편집
- 곡 카드 우클릭 → **정보 수정**에서 제목·아티스트·KEY/BPM·숙련도·난이도 등을 편집하고 **저장**합니다.
- 저장 후 창이 닫히고 라이브러리에 바로 반영됩니다.
